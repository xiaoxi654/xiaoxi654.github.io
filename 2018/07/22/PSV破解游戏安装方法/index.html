PSV破解游戏安装方法

发表于 2018-07-22 | 更新于 2018-07-26 | 分类于 游戏 |

前言

群上既然有人说要psv破解游戏的安装方式，那我就写一个好了

PSV平台游戏安装

vpk格式游戏的安装方式

此格式安装方式最简单，但分成两种安装方式

直接安装

  1. 把vpk文件直接复制到psv上
  2. 直接打开vitashell进行安装

文件夹格式安装

此方式相对于直接安装，可以节省空间

  1. 把vpk文件后缀名改为zip
  2. 解压这个vpk
  3. 按照下面maidumptool格式的游戏进行安装即可

MaiDumpTool格式游戏的安装方式

  1. 将游戏的文件夹名改为游戏ID（例如 PCSH00088）
  2. 在psv上安装MaiDumpTool
  3. 打开一次 MaiDumpTool
  4. 将游戏文件夹直接传进psv的 ux0:mai 目录里面
  5. 打开 MaiDumpTool 选择安装游戏即可

NoNpDrm游戏的安装方式

这种格式的优势在于可以直接在psv内更新，而且任意正版数字版游戏均可直接使用

  1. 在psv内安装NoNpDrm插件
  2. 直接将游戏文件夹传进psv的 ux0:app 里
  3. 在vitashell里，回退到选择分区的页面，选择 Refresh LiveArea™

使用pkgj安装游戏

这种方式的优势是可以直接在psv上安装游戏，但依赖于NoNpDrm

  1. 将NoNpDrm插件安装到你的psv上
  2. 安装pkgj

  4. 在psv上打开pkgj

  5. 按下三角，选择Refresh
  6. 直接找你想要的游戏，安装即可

PSM平台游戏安装

安装游戏

  1. 安装NoPsmDrm插件
  2. 将PSM游戏放进ux0:psm里面
  3. 关掉psv
  4. 按住电源键+PS键+R键进行开机，进入安全模式
  5. 选择重新建立数据库
  6. 重启后就可以在PSV的LiveArea™看到游戏的气泡了

安装PSM运行包

这时游戏还无法启动，提示需要安装PSM运行包

  1. 在电脑上下载Qcma和psvimgtools-frontend
  2. 打开Qcma，psv连接到电脑
  3. 打开psvimgtools
  4. 选择Easy Installers
  5. 选择PSMRuntime，点击Install
  6. 在psv上打开内容管理，选择个人电脑→本主机→应用程式→PS Vita
  7. 安装PSM运行包就可以了

特殊事项

  • 使用了卡套的用户需要在SD卡和原装记忆棒里都放一份游戏
  • 使用卡套的用户需要在重新启动后进入vitashell，选择一次Refresh LiveArea™才可以重新看到卡套内的游戏
  • 部分游戏（例如Cytus Lambda）进入游戏后无法联网，无法游玩一些游戏内容
